Enhancement Two: Algorithms and Data Structure

The second enhancement to the Grazioso Salvare application involved optimizing data retrieval processes by leveraging advanced data structures, specifically hash maps. This modification built upon the foundational work established in the first enhancement, aiming to enhance the efficiency and performance of the system.

About the Enhancement

The implementation of hash maps in the application represented a significant improvement in search functionality. By transitioning from linear search methods to hash maps, the time complexity of search operations was reduced from O(n) to O(1) under most circumstances. This enhancement was critical for scaling the application and improving user experience by providing faster access to rescue animal records.

Alignment with Course Outcomes

This enhancement aligns with the course outcome of "Design and evaluate computing solutions that solve a given problem using algorithmic principles and computer science practices and standards appropriate to its solution, while managing the trade-offs involved in design choices." The strategic use of hash maps to optimize search functionality exemplifies the application of algorithmic principles to solve a critical performance issue within the application.

Challenges and Learnings

A challenge encountered during this enhancement was ensuring the integrity of data while transitioning to a hash map-based storage mechanism. This required a thorough understanding of hash map operations and careful planning to prevent data collision and ensure efficient data retrieval. The process was an invaluable learning opportunity in data structure optimization and its impact on software performance.

Skills Demonstrated

Through this enhancement, I demonstrated proficiency in data structures and algorithms, particularly in applying hash maps to improve system performance. Additionally, this enhancement showcased my ability to analyze and optimize existing code for better efficiency and scalability.
